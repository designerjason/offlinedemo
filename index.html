default web page - you're online

Your normal website will be displayed here if there's a network connection. If you go offline and reload the page, you should be taken to the offline web app. Most browsers support it, but some do not (mainly older versions of Internet Explorer). Just ensure you're on the latest browser version.

The browsers that do support the offline web app are as follows:

  • Microsoft Edge (v17+)
  • Firefox (v44+)
  • Google Chrome (v45+)
  • Safari (v11.1+)
  • iOS Safari (v11.3+)
  • Samsung Internet (v4+)

If a browser does not support this feature, you'll just see your default browser message telling you that you have no internet connection.
